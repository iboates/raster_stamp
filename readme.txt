Thank you for using the Raster Stamp tool.

ArcGIS Desktop instructions:

1. Load the toolbox into ArcToolbox.
2. Open the Raster Stamp tool.
3. Input the parameters (some sample data has been provided in sample_data.gdb

Python instructions:

1. Put the file in /src/raster_stamp.py into the same folder as your script.
2. Import it with the command "from raster_stamp import raster_stamp"
3. Call the function with the command (rasterstamp(params)).
